# Log Analyzer CLI

![Version](https://img.shields.io/badge/version-1.0.1-blue)

A Python command-line tool for parsing, filtering, summarizing, and exporting log files. Designed to handle multiple log formats with regex support and user-configurable preferences.

## Features

- Parse and analyze log files with support for multiple formats
- Filter logs by level, date range, and more
- Summarize log data with counts per log level
- Export filtered logs to CSV or JSON
- Interactive CLI mode with command loop and ASCII art banner
- Configurable user preferences saved via a config file
- Clean OOP design and modular code structure
